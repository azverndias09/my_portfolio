# Azvern Dias Portfolio

Personal developer portfolio built with React, TypeScript, Vite, and Tailwind CSS.

## Overview

This repository contains the source for my portfolio website, highlighting experience, selected projects, and achievements.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- lucide-react icons

## Scripts

```bash
npm run dev      # start local dev server
npm run build    # type-check and build for production
npm run preview  # preview production build locally
```

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Deployment Notes

- `CNAME` is included for custom domain mapping.
- Static assets are served from `public/`.
